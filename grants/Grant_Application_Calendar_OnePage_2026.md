# One-Page Grant Application Calendar (2026)

## March
- Contact NRCS and request conservation planning visit
- Build scope and map: tunnel, irrigation, drainage, soil practices
- Capture baseline photos + KPI baseline

## April
- Finalize quotes and draft EQIP package
- Submit VAPG only if value-added plan is mature
- Build REAP energy concept sheet

## May-June
- Submit/refresh EQIP package for ranking windows
- Organize compliance records (UEI/SAM as needed)
- Build CSP readiness documentation

## July-August
- Draft NC AgVentures narrative and budget
- Confirm 15% cash match source
- Get Extension review

## September-October
- Submit AgVentures during cycle window
- Submit REAP if technical scope is complete
- Recheck EQIP/CSP deadlines

## November-December
- Submit SARE only if true research/demo design
- Publish annual outcomes summary
