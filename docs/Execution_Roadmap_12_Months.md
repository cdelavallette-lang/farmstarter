# 12-Month Execution Roadmap

## Quarter 1 (Mar-May)
- Soil tests by 3-5 zones
- NRCS site visit and conservation plan intake
- Install/repair primary water + drip on active blocks
- Launch weekly buyer outreach

## Quarter 2 (Jun-Aug)
- Implement first infrastructure phase (irrigation/drainage/tunnel components)
- Run weekly scouting + KPI tracking
- Prepare AgVentures draft before cycle

## Quarter 3 (Sep-Nov)
- Fall production push (greens, roots, herbs)
- Submit grant applications by cycle windows
- Publish monthly financial and KPI updates

## Quarter 4 (Dec-Feb)
- Winter protected production planning
- Year-end reporting for grants and partners
- Refresh crop and cashflow plans for next cycle
