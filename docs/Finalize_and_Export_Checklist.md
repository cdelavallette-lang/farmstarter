# Finalize and Export Checklist

## 1) Fill your farm profile once
1. Copy [config/farm_profile.env.example](../config/farm_profile.env.example) to `config/farm_profile.env`
2. Edit values with your real contact info.
3. Run:
   - `bash scripts/fill_profile.sh`

- [grants/Public_Grant_Packet_27312.md](../grants/Public_Grant_Packet_27312.md)

## 3) Export to PDF (quick method)
Use VS Code Markdown preview:
1. Open file.
2. Open preview.
3. Print -> Save as PDF.

## 4) Google Site publish package
