# Publish Your Website for Free (Fastest Path)

## Option A (Fastest): Netlify Drop (Free, 5 minutes)
1. Open this folder in your file explorer: [site/public](public)
2. Go to https://app.netlify.com/drop
3. Drag and drop the entire [site/public](public) folder onto that page.
4. Netlify instantly gives you a live URL.
5. (Optional) Click **Site settings** -> **Change site name** for a cleaner URL.

Your homepage file is: [site/public/index.html](public/index.html)

## Option B: GitHub Pages (Free, more durable)
1. Create a new GitHub repository named `santa-puala-farm-site`.
2. Upload contents of [site/public](public) to the repo root.
3. In GitHub: **Settings** -> **Pages**.
4. Set Source to **Deploy from a branch**, branch `main`, folder `/root`.
5. Save and wait 1-3 minutes.
6. Your site URL appears in Pages settings.
